# AutoAccepter

AutoAccepter is a Python script that automatically accepts game ready checks for you. This can be useful for games that require you to manually accept a ready check before starting a match.

## Features

- Automatically detects and accepts game ready checks
- Lightweight and easy to use

## Requirements

- Python 3.x
- `pyautogui` library
- opencv-python

## Installation

1. Clone the repository:
    ```
    git clone https://github.com/yourusername/autoaccepter.git
    ```
2. Navigate to the project directory:
    ```
    cd autoaccepter
    ```
3. Install the required dependencies:
    ```
    pip install -r requirements.txt
    ```

## Usage

1. Run the script:
    ```
    python main.py
    ```
2. The script will start monitoring for game ready checks and automatically accept them.
